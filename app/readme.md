### 主要功能
1. **图像上传**：用户可以上传多张肾脏病理图像，支持多种格式（如 JPG、PNG）。
2. **模型调用**：通过预训练的深度学习模型对上传的图像进行分析，包括分割、检测和分类。
3. **数据统计**：对图像和患者的基本信息进行统计分析，包括年龄、性别、疾病类型等。
4. **结果可视化**：将分析结果以图像和统计表格的形式展示，便于用户理解和使用。

### 数据结构
- **annotations**：图像的分割、检测和分类的标注信息。
- **imgs**：以病人为单位存放的原始图像数据。
- **reports**：病人对应的病理诊断报告。
- **tags**：数据的标签信息，包括病人病理类型、图像标尺放大倍率等。

### 技术栈
- 使用 Python 语言，结合 Streamlit 框架构建用户界面。
- 利用深度学习模型进行图像分析，使用 OpenCV 和 NumPy 进行图像处理。

### 使用说明
1. 启动应用后，用户可以在侧边栏上传图像。
2. 点击"运行模型"按钮，系统将对上传的图像进行分析。
3. 分析结果将显示在页面上，包括图像的处理结果和相关统计信息。

这个项目旨在为研究人员和临床医生提供一个高效的工具，以便于对肾脏病理图像进行深入分析和研究。

- 运行webUI命令
streamlit run app/gui.py --server.enableXsrfProtection=false
